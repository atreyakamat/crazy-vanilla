# Crazy Vanilla

A fresh React application built with Vite.

## Getting Started

### Install dependencies
```bash
npm install
```

### Run development server
```bash
npm run dev
```

### Build for production
```bash
npm run build
```

### Preview production build
```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- Modern ES6+ JavaScript
